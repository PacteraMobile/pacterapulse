Pactera Pulse API

Welcome to the Pactera Pulse API!

  • To use the service, please post the result to vote/emote with parameters timestamps and value;
  • Parameter timestamp format is yyyy-MM-dd'T'HH:mm:ss;
  • Parameter value range is "happy", "neutral", "sad";
  • Here is the example post url: http://localhost:8080/vote/emotion?timestamp=ssss&value=happy
  • The response is json {"status", "ok"};
